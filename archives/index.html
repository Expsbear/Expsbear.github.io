    提交到码云上

    1. 安装Git，在本地创建一个项目文件夹，在该文件夹中右击打开Git Bash Here选项，在命令行窗口输入git init初始化库管理窗口。

    2. 修改Git的全局配置（下次不需修改）

    git config –global user.name “你在码云的用户名”
    git config –global user.email “你在码云的邮箱”
