用hexo+github快速搭建个人博客

  • Expsbear

安装node.js

这里以windows环境为例，到node.js官方，选择合适的版本进行下载并安装。

安装git客户端

到git官网选择合适的版本进行下载，注意勾选Git Bash Here和Git GUI Here选项，方便随时随地打开命令窗口。

安装hexo

在电脑上创建一个文件夹，创建后进入文件夹中，按住shift键，右击鼠标点击“在此处打开Powershell窗口”，使用npm命令安装hexo，输入：

npm install -g hexo-cli

初始化hexo

接着输入以下命令初始化hexo：

npm init
安装依赖包:
npm install
