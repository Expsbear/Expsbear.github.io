什么是this?

首先，我们来看一段代码：

在上面的代码中，obj.foo()和foo()虽然都指向了同一个函数，但结果却不一样，原因就在于this关键字，this指的是函数运行时所在的环境。对于obj.foo来说，foo运行在obj环境，因为foo左边是obj对象调用的，所以this指向obj；而对于foo()来说，foo运行在全局环境，因为foo指向了函数本身，没有对象调用了foo，所以this指向全局。总结出来下面一句话：

this 的值并不是由函数定义放在哪个对象里面决定，而是函数执行时由谁来唤起决定。
