闭包

什么是闭包？闭包是指能够访问自由变量的函数。换句话说，在闭包中定义的函数可以”记忆”它被创建的环境。(注：自由变量是既不是在本地声明也不作为参数传递的一类变量，也可以这么说，如果一个作用域中使用的变量不是在该作用域中声明的，那么这个变量对于该作用域来说就是自由变量。)

在上面的例子中，foo函数创建了一个局部的自由变量num和bar函数，bar函数没有自己的局部变量，但是由于使用了闭包，它可以通过外部函数访问num变量。

下面再来看一个例子： 

function sayHello(){
    var say = function(){
        console.log(hello)
    }
    var hello = 'hello world';
    return say
}
var say = sayHello();
say()      //hello world
